Markdown 零基础教程（纯文字版）

1. 什么是 Markdown

  • 用简单符号写排版
  • 不用点鼠标调格式
  • 笔记、文档、公众号、GitHub 都能用

2. 最常用语法（背会这 9 个就够）

1）标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

2）加粗

**这是加粗文字**

3）斜体

*这是斜体*

4）删除线

~~这是删除线~~

5）无序列表

- 第一项
- 第二项
- 第三项

6）有序列表

1. 第一项
2. 第二项
3. 第三项

7）引用

> 这是一段引用文字

8）代码

  • 行内代码：
`这里放代码`
  • 代码块：
```python
print("hello")
```

9）链接

[显示文字](链接地址)

10）图片

![图片说明](图片地址)
